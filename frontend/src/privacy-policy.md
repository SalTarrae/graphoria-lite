# Privacy Policy

Last updated: 2026

---

## 1. Introduction

This Privacy Policy describes how the Graphoria Lite application ("the Application") collects, uses, and protects user data.

Graphoria Lite is an interactive web-based system for visual modeling of interconnected data structures. The application is developed 
for educational purposes as part of a software engineering coursework for the course "Software Standardization and Documentation" at the 
Zhytomyr Polytechnic State University.

---

## 2. Data Collection

The Application does not collect or store personal data on a server.

However, the following types of data may be processed locally in the user's browser:

* Graph data (nodes and connections created by the user)
* User interface preferences
* Cookie consent settings

All such data is stored locally using browser storage mechanisms (e.g., LocalStorage).

---

## 3. Cookies Usage

The Application uses cookies or similar technologies to:

* Ensure proper functionality of the application
* Store user preferences regarding cookie consent

The following categories of cookies are supported:

* **Necessary cookies** — required for basic functionality (always enabled)
* **Analytics cookies** — used for usage analysis (optional)
* **Marketing cookies** — used for advertising purposes (optional)

Users can manage their preferences through the cookie consent popup displayed on first visit.

---

## 4. Legal Basis (GDPR)

The processing of data is based on:

* User consent (Article 6(1)(a) GDPR)
* Legitimate interest for essential functionality (Article 6(1)(f) GDPR)

Users have the right to withdraw consent at any time by clearing browser storage.

---

## 5. Data Storage

All data is stored locally in the user's browser and is not transmitted to any external server.

The backend server used in this project operates only as a demonstration API and does not persist personal user data.

---

## 6. Data Sharing

The Application does not share user data with third parties.

No personal data is sold, transferred, or disclosed.

---

## 7. User Rights

Under GDPR, users have the right to:

* Access their data
* Correct their data
* Delete their data
* Withdraw consent

Since all data is stored locally, users can exercise these rights by:

* Clearing browser storage
* Resetting cookie preferences

---

## 8. Data Security

The Application follows standard development practices to ensure data safety.

However, as data is stored locally, users are responsible for maintaining the security of their devices and browsers.

---

## 9. Third-Party Libraries

The Application uses open-source libraries (e.g., React, React Flow, Express).

All dependencies have been analyzed using license-checker, and no restrictive licenses were identified.
